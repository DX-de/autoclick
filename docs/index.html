Auto Clicker Pro

Clic automatique simple et puissant pour Windows et Linux. Raccourcis clavier, positions enregistrées, répétition infinie.

🐧 Télécharger Linux 🪟 Télécharger Windows
Voir sur GitHub

Version 1.0.0 · Aucune inscription requise

⚡

Rapide à configurer

Régle les clics par seconde, choisis gauche/droit/molette, et c'est parti.

⌨️

Raccourcis globaux

F6 pour démarrer/arrêter, Échap pour l'arrêt d'urgence. Personnalisables.

📍

Positions enregistrées

Capture plusieurs points à cliquer en séquence ou aléatoirement.

🔒

100 % local

Aucune connexion internet requise. Tes réglages restent sur ton PC.

Installation

Linux
  1. Télécharge le fichier .zip
  2. Dézippe-le
  3. Dans un terminal : chmod +x AutoClickerPro puis ./AutoClickerPro

Windows
  1. Télécharge AutoClickerPro.exe depuis la release
  2. Double-clique pour lancer
  3. Si l'antivirus bloque : c'est un faux positif courant avec PyInstaller
⚠️ Utilise cet outil de manière responsable. L'automatisation peut être interdite sur certains jeux ou plateformes. Usage personnel uniquement.
