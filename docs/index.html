Installation

🐧 Linux

  1. Télécharge le .zip
  2. Dézippe le fichier
  3. Terminal : chmod +x AutoClickerPro
  4. Puis : ./AutoClickerPro

🪟 Windows

  1. Télécharge AutoClickerPro.exe
  2. Double-clique pour lancer
  3. Antivirus : faux positif possible avec PyInstaller — autorise l'app
⚠️ Utilise cet outil de manière responsable. L'automatisation peut être interdite sur certains jeux ou plateformes.
